# 와프(WAF) 회사소개 웹사이트 디자인 아이디어

## 아이디어 1: 미니멀리즘 + 기술 포커스 (확률: 0.08)

**디자인 무브먼트**: 모던 미니멀리즘 (Modern Minimalism)

**핵심 원칙**:
- 깔끔한 라인과 충분한 공백으로 기술력 강조
- 단색 + 1-2개 악센트 컬러로 전문성 표현
- 데이터 시각화와 기술 다이어그램을 주요 요소로 활용
- 타이포그래피의 계층구조로 정보 우선순위 명확화

**색상 철학**:
- 베이스: 깊은 네이비(#0F172A) + 순백(#FFFFFF)
- 악센트: 테크 블루(#0EA5E9) + 라임 그린(#84CC16)
- 철학: 신뢰성과 혁신성을 동시에 표현

**레이아웃 패러다임**:
- 비대칭 그리드 시스템: 좌측 텍스트 + 우측 시각화
- 섹션별 다양한 배치로 단조로움 방지
- 수직 리듬감 있는 스페이싱

**시그니처 요소**:
- 기술 다이어그램: CNC, MES, AI-PC 아키텍처 시각화
- 미니멀 아이콘: 라인 스타일의 커스텀 아이콘
- 데이터 카드: 성과 지표를 카드 형태로 표현

**인터랙션 철학**:
- 호버 시 미묘한 색상 변화와 스케일 효과
- 스크롤 트리거 애니메이션으로 정보 점진적 노출
- 클릭 가능 요소는 명확한 피드백 제공

**애니메이션**:
- 진입 애니메이션: fade-in + slide-up (200ms)
- 호버 효과: 색상 전환 (150ms) + 스케일 1.02
- 스크롤 애니메이션: 요소별 stagger 효과

**타이포그래피**:
- 디스플레이: Poppins Bold (700) - 헤더, 제목
- 바디: Inter Regular (400) - 본문
- 악센트: Poppins SemiBold (600) - 강조 텍스트

## 아이디어 2: 다이나믹 + 산업 감성 (확률: 0.07)

**디자인 무브먼트**: 인더스트리얼 디자인 (Industrial Design)

**핵심 원칙**:
- 기계적 미학: 그리드, 정밀한 라인, 기술적 디테일
- 어두운 배경에 밝은 악센트로 에너지 표현
- 기계 부품, 회로도 같은 시각 모티프 활용
- 움직임과 변화로 역동성 표현

**색상 철학**:
- 베이스: 차콜 그레이(#1A1A1A) + 라이트 그레이(#F5F5F5)
- 악센트: 오렌지(#FF6B35) + 전기 블루(#00D9FF)
- 철학: 산업의 견고함과 기술의 활기

**레이아웃 패러다임**:
- 대각선 컷 섹션: 각진 경계로 시각적 긴장감
- 오버래핑 요소: 깊이감 있는 레이어링
- 비대칭 그리드로 동적 흐름 생성

**시그니처 요소**:
- 기하학적 패턴: 육각형, 삼각형 배경
- 기술 라인: SVG 애니메이션 라인
- 산업 이미지: CNC 기계, 제조 현장 사진

**인터랙션 철학**:
- 클릭 시 전기적 피드백 (번쩍임 효과)
- 호버 시 색상 반전 또는 글로우 효과
- 스크롤 시 요소 회전 및 변형

**애니메이션**:
- 진입: zoom-in + rotate (300ms)
- 호버: glow 효과 + 색상 변화 (200ms)
- 스크롤: parallax + 요소 변형

**타이포그래피**:
- 디스플레이: Bebas Neue (700) - 강렬한 헤더
- 바디: Roboto (400) - 기술적 느낌
- 악센트: Roboto Mono (600) - 코드 같은 강조

## 아이디어 3: 프리미엄 + 신뢰 (확률: 0.06)

**디자인 무브먼트**: 럭셔리 미니멀리즘 (Luxury Minimalism)

**핵심 원칙**:
- 고급스러운 여백 활용으로 프리미엄 느낌
- 부드러운 곡선과 우아한 타이포그래피
- 신뢰성을 위한 일관된 구조와 명확한 정보 계층
- 섬세한 디테일: 미묘한 그림자, 부드러운 그래디언트

**색상 철학**:
- 베이스: 크림(#FAFAF8) + 깊은 슬레이트(#2D3E50)
- 악센트: 골드(#D4AF37) + 포레스트 그린(#1B4332)
- 철학: 전통적 신뢰성과 현대적 혁신의 조화

**레이아웃 패러다임**:
- 대칭적 구조로 안정감 표현
- 넓은 마진과 여백으로 우아함 강조
- 섹션 구분이 명확하되 부드러운 전환

**시그니처 요소**:
- 골드 악센트 라인: 섹션 구분 및 강조
- 우아한 일러스트: 제조 과정의 세련된 표현
- 타이포그래피 강조: 큰 서체로 메시지 임팩트

**인터랙션 철학**:
- 부드러운 전환: 모든 상태 변화가 자연스럽고 우아함
- 호버 시 미묘한 엘리베이션 효과
- 클릭 피드백: 부드러운 색상 변화

**애니메이션**:
- 진입: fade-in + subtle slide (250ms, ease-out)
- 호버: 그림자 강화 + 스케일 1.01 (150ms)
- 스크롤: 부드러운 fade-in-up (ease-in-out)

**타이포그래피**:
- 디스플레이: Playfair Display (700) - 우아한 헤더
- 바디: Lato (400) - 가독성 높은 본문
- 악센트: Lato SemiBold (600) - 세련된 강조

## 선택된 디자인: 아이디어 1 - 미니멀리즘 + 기술 포커스

**선택 이유**: 와프의 기술력과 전문성을 가장 효과적으로 표현하면서도, IT·OT 통합 솔루션의 복잡성을 명확하게 시각화할 수 있기 때문입니다. 깔끔한 미니멀 디자인은 기술 기업의 신뢰성을 강조하고, 데이터 시각화 요소는 제조 혁신이라는 핵심 메시지를 직관적으로 전달합니다.

**디자인 철학 요약**:
- 깊은 네이비 배경 + 순백 텍스트로 전문성 표현
- 테크 블루 악센트로 혁신성 강조
- 기술 다이어그램과 미니멀 아이콘으로 정보 시각화
- 비대칭 그리드로 동적이면서도 정돈된 레이아웃
- 부드러운 애니메이션으로 현대적 느낌 표현
